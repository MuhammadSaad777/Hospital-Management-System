 Medicore Hospital Management System

 OverView:

1. A  hospital management system made using C++ and SFML for GUI
2. Used to manage patient,doctors,admins,appointments,bills records.
3.User friendly GUI to perform any task with files appending and updating on basis of actions.


Features:
1. Allowing patients to book,cancel, view appointments and manage bills.
2. Allowing doctors to view todays appointments and mark their status also prescribing patients.
3. Allowing admin to manage system by adding, removing doctors and patients and generating reports.
4. Login system with privacy respect so data is not misused by others.

Technologies used:
1. C++
2. OOP concepts with headers and cpps
3. Attractive User interface
4.FileHandling 

Project Structure:
1. Main creates objects and calls displaycompletemenu
2. Menu file handles etire sfml and button logics making updates in files accordingly
3. Several other supporting files are used as well such as patients, doctors, admins.

Running procedure:
1.Clone following github repository:

2. Open project in VisualStudio2026
3. Compile project and run it

1.Simple management system helping understand 
  a.OOP Concepts
  b.PF Concepts
  c.File Handling Concepts
